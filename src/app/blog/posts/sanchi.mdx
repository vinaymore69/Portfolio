## Overview

Sanchi is a comprehensive, multi-lingual web platform developed for the Vidyalankar Volunteering Committee. The platform is focused on creating positive social impact by encouraging student participation in volunteering activities across eight major social domains.

It serves as a centralized hub for the committee’s operations, offering public information, educational resources, and a full user management system. The platform supports English, Hindi, and Marathi to ensure accessibility for a wider audience and is built on a modern full-stack architecture with a Node.js backend.

## Key Features

### User System

- Secure user registration, login, and password reset functionality  
- Personal dashboard for volunteers  
- Profile management and account updates  

### Multi-Language Support

- Dynamic content switching between English, Hindi, and Marathi  
- Language preference stored using cookies  
- Implemented through JavaScript-based content mapping  

### Social Domains

Dedicated pages for eight focus areas:

- Girls Education  
- Primary Education  
- Unprivileged Students Welfare  
- Physical Health  
- Mental Health  
- Environment  
- Digital Literacy  
- Anti-Narcotics  

### Resource Hub

- Audiobooks with a custom synchronized player  
- Educational video library  
- Notes downloader with filtering by class and subject  

### User Dashboard

- Profile editing interface  
- Upload forms for audio, video, and notes  
- Digital sticky notes application with full CRUD operations  
- Notes stored per user account  

### Dynamic Public Pages

- Home page with advanced animations and interactive domain list  
- About page with team profiles and contribution slider  
- Responsive gallery with masonry layout and lightbox  
- Contact form with multiple file attachment support  

### Backend Services

- Automatic video processing into HLS streams with thumbnail generation  
- File upload handling for media and documents  
- Email services for contact submissions and password resets  

## Technology Stack

### Frontend

- HTML5 for structure  
- CSS3 (Flexbox, Grid, Media Queries) for layout and responsiveness  
- JavaScript (ES6+) for interactivity and API communication  

Libraries and Tools:

- GSAP for animations  
- Swiper.js for sliders and carousels  
- Lenis for smooth scrolling  

### Backend

- Node.js for server-side runtime  
- Express.js for routing and APIs  
- PostgreSQL for data storage  
- Nodemailer for email services  
- Multer for file uploads  
- FFmpeg for video transcoding and thumbnail extraction  

### Integrations

- Botpress chatbot for user assistance  

## Project Structure

The project is organized into modular folders for maintainability:

- **html/** — All static pages including public pages, domain pages, resource pages, and authentication interfaces  
- **css_files/** — Global styles, page-specific styles, domain styles, and dashboard styles  
- **javascript_files/** — Core scripts, page logic, language system, dashboard scripts, and utilities  
- **custom_modules/** — Backend modules for video processing, file uploads, email services, and helper utilities  

## HTML Pages
